# Descripción del proyecto

<p>Este repositorio pertenece al backend de mi mini-eccommerce llamado Madison Store. Fue hecho con el fin practicar y afianzar conocimientos de backend.</p>
<p>Dejo el link del repositorio del frontend de mi proyecto por si queres echarle un vistazo.</p>

\## 📁 Acceso al proyecto

<p>Podes clonar mi repositorio.</p>

\## 🛠️ Abre y ejecuta el proyecto

 \#1 Utilizar el comando `npm install` para instalar dependencias.
 \#2 Crear archivo .env, usando como referencia el .env.template
 \#3 Para poder correr el backend utilizar el comando `npm run dev`.



# Tecnologias Utilizadas

- Node.js + Express
- TypeScript
- MongoDB + Mongoose
